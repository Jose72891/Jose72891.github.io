# Product Requirements Document (PRD)
**Project:** "Joseph Pulla"
**Owner / Author:**  Joseph Pulla
**Date:** 2025-11-5
**Version:** 1.0

---

## 1. Overview
**Website Name:** Joseph Pulla  
**Purpose:** To provide info from my personal lifestyle that would interest other students so that they would choose to study about what I have been learning. The major that I was studying is "Computer Information Systems"; I had to work on several projects that involves computing. 

**Target Audience:** NYC residents, computer experts and scientists looking for students who has great knowledge of computer information systems

## 2. Goals
- Provide great info that would catch students attention about computer information systems
- Present the projects that I had already done and put it on my portfolio
- Present the computing skills that I already experienced

## 3. Features
1. **Home Section**  
   - Welcome message and home page about Joseph Pulla "Portfolio"
   - Overview summary about my daily college lifestyle
   - 
   - Great image of the major "Computer Information System"

2. **Skills Section**  
   - Types of skills that I was able to gain when doing the computing projects 
   - Images of the computing projects

3. **Portfolio Section**
   - Intro about my goal that I am trying to reach when studying computer information systems
   - Pics of the work that I had done
   - Catch the audience attention
   - Get students to be interested in the major

4. **About Section**  
   - Story of why I chose the major "Computer Information System" when I started college and how it can be challenging sometimes 
   - Highlighting any achievements, skills or experiences that I already achieved  

5. **Contact Section**  
   - Personal phone number and email
   - Social media links: LinkedIn, Facebook and Whatsapp
   - Zoom meeting link 


## 4. User Stories
- **As a freshmen student**, I want to view the table section on Joseph's portfolio so that I can look on the work that I had already done so that me and other students can learn more about computing.  
- **As a social media user**, I want to post pictures of projects that I had done so that people can recommend it and see how interesting it is.
- **As a web developer**, I want to view the table and about section on Joseph's portfolio so that I would see how interesting computer information systems is and to work on a project that is similar to what Joseph has done like coding or designing websites.

## 5. Design 
- **Brand aesthetic** Minimal, modern aesthetic with clean typography and ample white space.  
- **Color Scheme:** Modernized Italian color scheme with accent: Blue, Red and Orange
- **Typography:** Garamond (updated Roman typeface) for headings, Times New Roman (sans-serif) for body text.  
- **Navigation:** Top menu bar with links to Tables (with dropdown for computing projects), About, and Contact.  
- **Layout:** Simple, single-column scrolling pages with sections clearly separated.  
- **Images:** High-quality photos of technologies and computing projects

## 6. Technical Stack / Tools
- Programming: Python, Java
- Sublime Text for website designs
- HTML5, CSS
- Optional: JS for interactivity (lightbox, smooth scroll)
- GitHub Pages for hosting
- Built with VSCode and GitHub Copilot assistance

## 7. Functional Requirements
- Navigation bar with smooth scrolling to sections.   
- Lightboxes (fullscreen views) for images of Joseph's projects 
- External links open in new tabs.
- Fully responsive (mobile phones, tablets and laptops).  
- Cross-browser compatible (Google Chrome, Safari and Firefox).

## 8. Accessibility / UX Notes
- Alt text for all images
- High contrast for text readability
- Alternative formats
