# Product Requirements Document (PRD)
**Project:** "Joseph Pulla"
**Owner / Author:**  Joseph Pulla
**Date:** 2025-12-3
**Version:** 1.0

---

## 1. Overview
**Website Name:** Joseph Pulla  
**Purpose:** To share my personal lifestyle in a portfolio style so that I would show people about my past college years, what major I studied and present past projects that would interest people and to understand the major. 

**Target Audience:** NYC residents, computer experts and scientists looking for students who has great knowledge of computer information systems

## 2. Goals
- Getting a good job that is based on my major
- Show the computer science experts about my computing skills
- Receiving a high amount of money

## 3. Features
1. **Home Section**  
   - Welcome Header page that says Joseph Pulla "Portfolio"
   - Introduction about who I am and I study Computer Information Systems
   - A good of myself for the portfolio home page

2. **Skills Section**  
   - Types of skills that I was able to gain when doing computing projects
   - Provide great tips to display the meaning for each skills

3. **Projects Section**
   - Present the computing projects that I had done in the past college years
   - Introduce of what each project is about
   - Images of the computing projects

4. **Extras Section**  
   - Videos for more info of Computer Information Systems
   - Articles for more info of CIS
   - Apps that would teach beginner students about the projects that I had already done

5. **Contact Me Section**  
   - Personal phone number and email
   - Social media links: LinkedIn, Facebook, Whatsapp and Tiktok


## 4. User Stories
- **As a freshmen student**, I want to view the project section on Joseph's portfolio so that I can look back on the work that I had already done and it would interest me in studying CIS.  
- **As a social media user**, I want to post pictures of projects that I had done so that people can recommend it and see how interesting it is.
- **As a web developer**, I want to view the Project and Extra section on Joseph's portfolio so that I would see how interesting computer information system is and to work on a project that is similar to what Joseph has done like coding or designing websites.

## 5. Design 
- **Brand aesthetic** Minimal, modern aesthetic with clean typography and ample white space.  
- **Color Scheme:** Modernized Italian color scheme with accent: Blue, Red and Orange
- **Typography:** Garamond (updated Roman typeface) for headings, Times New Roman (sans-serif) for body text.  
- **Navigation:** Top menu bar with a with dropdown for Home, Projects, Skills, Extras and Contact Me.  
- **Layout:** Simple, single-column scrolling pages with sections clearly separated.  
- **Images:** High-quality photos of myself and computing projects

## 6. Technical Stack / Tools
- Programming: Python, Java
- Sublime Text for website designs
- HTML5, CSS
- Optional: JS for interactivity (lightbox, smooth scroll)
- GitHub Pages for hosting
- Built with VSCode and GitHub Copilot assistance

## 7. Functional Requirements
- Navigation bar with smooth scrolling to sections.   
- Lightboxes (fullscreen views) for images of Joseph's projects 
- External links open in new tabs.
- Fully responsive (mobile phones, tablets and laptops).  
- Cross-browser compatible (Google Chrome, Safari and Firefox).

## 8. Accessibility / UX Notes
- Alt text for all images
- High contrast for text readability
- Alternative formats
